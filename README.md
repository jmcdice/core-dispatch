# core-dispatch

## Sound and mic config

Make sure you've set both speaker and mic to the max using alsamixer

## Configure
```bash
  python -m venv .venv
  source .venv/bin/activate
  pip install .
```

## Source env vars
```bash
  source env.sh
```

## Run the reciever
```bash
  ./start_reciever.py
```
